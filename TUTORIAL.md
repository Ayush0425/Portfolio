## 4. GitHub Workflow (How to Save & Update)

You are using **Git** for version control. Here is the exact workflow to save your changes and update your live website.

### Step 1: Status Check
Check what files have changed:
```bash
git status
```

### Step 2: Stage Changes
Prepare all modified files for saving:
```bash
git add .
```

### Step 3: Save (Commit)
Save the snapshot of your code with a message explaining what you did:
```bash
git commit -m "Description of what I changed"
```

### Step 4: Upload (Push)
Send your saved changes to GitHub. This usually triggers your hosting provider to update the live site.
```bash
git push origin main
```
